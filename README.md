# OpenCart Auto Deploy

Автоматичне встановлення OpenCart на Ubuntu Server з Apache.
Підтримує reverse proxy (HTTPS зовні).

## Використання

```bash
git clone https://github.com/deputatN1/server-scripts.git /opt/opencart
cd /opt/opencart
chmod +x /opt/opencart/deploy.sh
cd /opt/opencart
sudo bash deploy.sh
